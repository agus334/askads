03

Cara Penggunaan

Kirim permintaan GET ke endpoint yang diinginkan, sertakan parameter berikut:

message: Isi permintaan kamu
04

Format Respon

Semua endpoint mengembalikan JSON dengan struktur:

{
  "status": 200,
  "creator": "ARR Official",
  "data": {
    "response": "Hasil respon permintaan kamu"
  }
}
05

Contoh Implementasi

Contoh penggunaan dalam handler bot:

case 'ai4': {
  if (!q) return reply(`Mau nanya apa?`)
  if (text.length > 500) return reply(`Maks. 500 karakter`)
  reply('Tunggu sebentar...')
  var res = await fetchJson(
    `https://api-lenwy.vercel.app/ai4chat?text=${q}`
  )
  await reply(res.data)
